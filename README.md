# AfriTalent - Plateforme de freelances tech en Afrique

## Nom et classe
**Mbacke Saliou** - Développement Web (Semestre 2)

## Description
AfriTalent est une plateforme fictive de mise en relation entre freelances tech et entreprises en Afrique. Ce site vitrine présente les services, les tarifs, et permet de découvrir les freelances disponibles. Le projet a été réalisé dans le cadre d'un projet pédagogique visant à démontrer la maîtrise de HTML5, CSS3, Bootstrap 5 et JavaScript vanilla.

## Technologies utilisées
- HTML5 (structure sémantique)
- CSS3 (Flexbox, Grid, Bento Grid, animations, variables CSS)
- Bootstrap 5 (navbar, grille, cards, carousel, accordion)
- JavaScript vanilla (DOM, événements, IntersectionObserver, validation formulaire)
- Git & GitHub (versioning, GitHub Pages)

## Fonctionnalités principales
1. **Dark Mode** avec persistance via localStorage
2. **Navbar dynamique** qui change de style au scroll
3. **Bouton retour en haut** avec smooth scroll
4. **Compteurs animés** avec IntersectionObserver
5. **Animations fade-in** des sections au scroll
6. **Filtrage dynamique** des freelances (sans rechargement)
7. **Validation de formulaire** (email regex, message 20+ caractères)
8. **Design responsive** (mobile, tablette, desktop)
9. **Carousel Bootstrap** pour les témoignages
10. **Accordion Bootstrap** pour la FAQ
