Dossier images - AfriTalent
===========================

But : vos images locales remplacent les visuels “placeholder”.

1) Mettez vos fichiers dans ce dossier `images/`
2) Utilisez exactement ces noms (par défaut) :

Accueil / index.html
 - hero.jpg
 - testimonial-amara.jpg
 - testimonial-kofi.jpg
 - testimonial-fatima.jpg

À propos / about.html
 - about-story.jpg
 - team-aicha.jpg
 - team-mamadou.jpg
 - team-fatou.jpg
 - team-ibrahim.jpg

Freelances / freelances.html (cartes générées)
 - freelance-01.jpg
 - freelance-02.jpg
 - freelance-03.jpg
 - freelance-04.jpg
 - freelance-05.jpg
 - freelance-06.jpg
 - freelance-07.jpg

 - freelance-09.jpg

Note
----
Si un fichier manque, le site garde automatiquement `placeholder.svg` (donc pas d’erreur visible).

Vous pouvez aussi modifier les chemins dans `js/site-images.js` si vous préférez d’autres noms/formats.
